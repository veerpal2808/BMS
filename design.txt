A. Client-Server Model
    
    [client - browser] -> <- [server - backend] -> <- [Database]

|   Client (Browser)|  ⇄   |  Server (Backend)   |  ⇄   |     Database     |

    1. Presentation Layer(HTML form)
    2. Logic Layer (Backend - future scope)
    3. Data Layer (Database - future scope)

  
|  Presentation Layer         |
|  (HTML Form - Frontend)     |

            ↓

|  Logic Layer                |
|  (Backend - contains Logic) |

            ↓

|  Data Layer                 |
|  (Database - Later)  |
